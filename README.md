## 📖 What is gateway?

gateway is a tool that connects your applications to OpenAI services. It runs in the background without a user interface (headless). It helps manage various users and controls access with security measures. It tracks usage costs and performance. It can also protect personal information automatically. You get one easy-to-run program with everything built-in. You set it up yourself, so your data stays under your control.

## 💻 System Requirements

- Windows 10 or later (64-bit recommended)
- At least 4 GB of RAM
- 500 MB of free disk space
- Internet connection for setups and API calls
- Administrator rights to install and run the program

## 🔧 Features at a Glance

- Works with multiple users securely
- Logs who used what and when
- Tracks how much you spend on AI calls
- Controls speed and number of requests
- Hides or removes personal information automatically
- One single program file to run
- Designed for easy audit and compliance checks

## 🔐 Security and Privacy Features

gateway includes tools to keep your data secure:

- User authentication to limit who can send requests
- Rate limits to prevent overload or abuse
- Optional hiding of personal information (PII redaction)
- Detailed logs for audit and review
- Cost tracking to know how much API use costs

It uses standard encryption and secure protocols to protect your connection.

## 📊 Monitoring and Logs

gateway records information on requests and responses:

- Track how many requests each user sends
- Log errors and warnings for troubleshooting
- See total cost over time for billing purposes
- Use simple text log files stored on your computer

Check the `logs` folder inside gateway’s working directory for these files.

## ⚠️ Common Troubleshooting Tips

- If gateway fails to start, make sure you downloaded the right `.exe` for Windows.
- Run the program as Administrator if you face permission errors.
- Check your internet connection, as gateway requires it for OpenAI access.
- If logs show API errors, verify your API keys and usage limits if you configured them.
- Reboot your computer if the program appears not to respond.

## 🛠 Updating gateway
